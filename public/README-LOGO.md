# Como adicionar o logo Aurelo

Para adicionar o logo da marca Aurelo ao sistema, siga estes passos:

1. Coloque a imagem do logo na pasta `public/` com um dos seguintes nomes:
   - `aurelo-logo.png` (recomendado)
   - `aurelo-logo.jpg`
   - `aurelo-logo.jpeg`
   - `aurelo-logo.webp`
   - `aurelo-logo.svg`

2. O componente tentará carregar automaticamente a imagem quando você reiniciar o servidor.

3. Se nenhuma imagem for encontrada, o sistema usará um SVG de fallback.

## Formatos suportados:
- PNG (recomendado para melhor qualidade)
- JPG/JPEG
- WEBP
- SVG

## Localização:
A imagem deve estar na pasta raiz `public/`, não em subpastas.
